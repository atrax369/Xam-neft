# Xam-neft
Azərbaycanın Xam Neft Sektoru: İllik Performans Analizi

Enerji sektoruna marağım olduğu üçün Azərbaycanın xam neft sektoru ilə bağlı kiçik bir analitik daşbord hazırladım. Məqsədim illər ərzində neft hasilatında və enerji göstəricilərində baş verən dəyişiklikləri daha aydın görmək idi.

Layihədə əsas olaraq xam neft hasilatı, ixrac göstəriciləri və ilkin enerji məhsullarının istehsalı kimi məlumatlara baxdım. Bu göstəricilərin illər üzrə necə dəyişdiyini daha aydın göstərmək üçün vizuallaşdırmalardan istifadə etdim.

Məlumatları Azərbaycan Respublikasının Dövlət Statistika Komitəsinin rəsmi statistika bazasından (stat.gov.az) götürdüm. Analiz əsasən 2007 və 2024-cü illərin məlumatları üzərində qurulub ki, dəyişiklikləri müqayisə etmək daha asan olsun.

Məlumatların təmizlənməsi və vizuallaşdırılması üçün isə Excel-dən istifadə etdim.

Bu layihə mənim üçün həm praktika, həm də enerji statistikası ilə işləmək baxımından maraqlı bir təcrübə oldu.
